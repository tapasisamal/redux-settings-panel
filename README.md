# Redux Settings Panel

A React application built with Redux Toolkit to demonstrate global state management for UI preferences such as theme and font size.

## Features

- Light / Dark theme toggle
- Dynamic global font size control
- Centralized state management using Redux Toolkit
- UI updates triggered via `useSelector`
- Side effects handled with `useEffect`

## Tech Stack

- React
- Redux Toolkit
- React-Redux
- Vite
- CSS

## Run Locally

```bash
git clone <https://github.com/tapasisamal/redux-settings-panel>
cd redux-settings-panel
npm install
npm run dev
```
